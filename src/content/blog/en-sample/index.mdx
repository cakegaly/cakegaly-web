## Code Block Example

Here's an example of a **TypeScript** code block:

```typescript
const greet = (name: string): string => {
  return `Hello, ${name}!`;
};

console.log(greet('World'));
```

You can also use **CSS** in a code block:

```css
body {
  background-color: #f0f0f0;
  font-family: 'Arial', sans-serif;
}

h1 {
  color: #333;
}
```
