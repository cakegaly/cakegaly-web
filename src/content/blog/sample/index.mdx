## コードブロックの例

こちらは **TypeScript** のコードブロックです：

```typescript
const greet = (name: string): string => {
  return `こんにちは、${name}！`;
};

console.log(greet('世界'));
```

次に **CSS** のコードブロックを紹介します：

```css
body {
  background-color: #f0f0f0;
  font-family: 'Arial', sans-serif;
}

h1 {
  color: #333;
}
```
