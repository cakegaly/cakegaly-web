## セットアップ手順

### ローカル開発環境を起動する

まずはじめに、GitHub から `cakegaly/cakegaly-web` を clone してください。

```sh
git clone git@github.com:cakegaly/cakegaly-web.git
cd cakegaly-web
```

次に、必要なパッケージを install して、`.env.local` を作成してください。

```sh
pnpm install
cp .env.example .env.local
```

ローカルでビルドしたあと、開発環境を起動してください。

```sh
pnpm build
pnpm dev
```

ブラウザで [`http://localhost:8888`](http://localhost:8888) を開いて、動作を確認してください。
